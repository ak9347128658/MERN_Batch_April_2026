Four ways to write functions in JavaScript with live examples

1
Function Declaration
most common — learn this first
function greet(name) {
  return "Hello, " + name + "!";
}

// calling it:
greet("Alice"); // "Hello, Alice!"
Try it live:
Hello, Alice!
Can be called BEFORE it is defined (hoisted)
2
Function Expression (stored in variable)
common — function stored in const
const multiply = function(a, b) {
  return a * b;
};

multiply(4, 5); // 20
Try it live:
×
multiply(4, 5) → 20
Must be defined BEFORE you call it
3
Arrow Function =>
modern — used EVERYWHERE in React & Node
// full version:
const square = (n) => {
  return n * n;
};

// short version (one line):
const square = (n) => n * n;

square(6); // 36
Try it live:
square(6) → 36
No function keyword. Shorter and cleaner.
4
Default Parameters
fallback when no value passed
function welcome(
  name = "Student",
  course = "MERN"
) {
  return `Hi ${name}, welcome to ${course}!`;
}

welcome(); // Hi Student, welcome to MERN!
welcome("Ali"); // Hi Ali, welcome to MERN!
Try it live — leave blank to use defaults:
Hi Student, welcome to MERN!
