Return statement animated explanation

function add(a, b) {
  let sum = a + b;
  return sum; // sends sum BACK to whoever called add()
}

let answer = add(3, 5); // answer gets the returned value
console.log(answer); // 8
Watch what happens step by step:
1. Call add(3,5) → 2. Enter function → 3. Run body → 4. return 8 → 5. answer = 8
