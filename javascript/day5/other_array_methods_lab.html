Other array methods includes indexOf find sort join slice forEach

includes( ) — does it exist?
Returns true or false
const fruits = ["mango","apple","banana"];
fruits.includes("apple") // true
fruits.includes("cherry") // false
—
indexOf( ) — find the position
Returns index number, or -1 if not found
fruits.indexOf("banana") // 2
fruits.indexOf("cherry") // -1 (not found)
—
find( ) — get first matching item
Returns the actual item (not index)
const nums = [10,25,8,30,5];
nums.find(n => n > 20) // 25
// returns FIRST item > 20
—
sort( ) — sort items
Strings: alphabetical. Numbers: need a compare function!
["banana","apple","mango"].sort()
// ["apple","banana","mango"]
[30,5,100,2].sort((a,b) => a-b)
// [2, 5, 30, 100]
—
join( ) — array to string
Glues all items into one string
["Alice","Bob","Charlie"].join(", ")
// "Alice, Bob, Charlie"
[1,2,3].join("-") // "1-2-3"
—
slice( ) — copy a portion
Doesn't change original — returns new array
const a = ["a","b","c","d","e"];
a.slice(1, 4) // ["b","c","d"]
// from index 1, up to (not including) 4
