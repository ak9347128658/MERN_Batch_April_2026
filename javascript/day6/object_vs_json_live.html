JavaScript object vs JSON comparison with stringify and parse live demo

JavaScript Object
const user = {
  name: "Ali",
  age: 22,
  active: true
};
✓ Keys don't need quotes
✓ Can have functions
✓ Values: any JS type
✓ Lives in memory
✕ Cannot be sent over internet as-is
JSON (string)
'{
  "name": "Ali",
  "age": 22,
  "active": true
}'
✕ Keys MUST have double quotes
✕ No functions allowed
✕ No undefined or NaN
✓ Just plain text (a string)
✓ Can be sent over internet!
