Object methods — functions stored inside objects with this keyword

const student = {
  name: "Ali Khan",
  score: 88,
  course: "MERN Stack",

  // method — a function INSIDE the object
  greet: function() {
    return `Hello! I am ${this.name}`; // 'this' = the object itself
  },
  getGrade: function() {
    return this.score >= 90 ? "A" : this.score >= 75 ? "B" : "C";
  },
  introduce: function() {
    return `I study ${this.course} and scored ${this.score}`;
  }
};
What is this? — Inside a method, this refers to the object itself. So this.name means student.name.
student.greet( )
Calls the greet method
student.greet() // call it!
—
student.getGrade( )
Calculates grade from score
student.getGrade() // A, B or C
—
student.introduce( )
Full introduction string
student.introduce() // full intro
