Interactive object creation and property access with dot and bracket notation

const student = {
  name: "Ali Khan",
  age: 22,
  city: "Hyderabad",
  course: "MERN Stack",
  passed: true,
  score: 88,
};
Click any property to access it:
Click a property to access it
Dot notation
most common — use this!
student.name // "Ali Khan"
student.age // 22
student.city // "Hyderabad"
—
Bracket notation
use when key is in a variable
student["name"] // "Ali Khan"
student["age"] // 22
let k = "city";
student[k] // "Hyderabad"
