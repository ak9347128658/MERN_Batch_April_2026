String concatenation and ternary operator interactive examples

String + operator concatenation
// + joins strings together
"Hello" + " World"
// → "Hello World"

// Template literal (backtick)
let name = "Alice";
`Hello ${name}!`
// → "Hello Alice!"
"Hello" + "World" → "HelloWorld"
Ternary ? : operator if-else shortcut
condition ? valueIfTrue : valueIfFalse
// same as if/else but one line!
let age = 20;
let msg = age >= 18
  ? "Adult"
  : "Minor";
age (20) >= 18 ? "Adult" : "Minor" → "Adult"
