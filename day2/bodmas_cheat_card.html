Golden Rules to Remember
Brackets always win When in doubt, add ( ) to force the order you want.
* / % before + - Same as school maths. Always.
Maths before comparison 5 + 3 > 2 * 4 → compute both sides first, then compare.
&& before || true || false && false → true || (false) → true
Same level = left to right 10 / 2 * 5 = (10/2)*5 = 25 — not 10/(2*5)=1
Assignment always last let x = 2 + 3 * 4 — right side fully calculated first, then stored.
