📡 REST API Reference ▼
GET /api/todos Fetch all todos
GET /api/todos/:id Fetch single todo by ID
POST /api/todos Create todo — body: { title, priority }
PUT /api/todos/:id Update todo — body: { title?, completed?, priority? }
DELETE /api/todos/:id Delete todo by ID
